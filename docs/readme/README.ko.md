# SanchoExo

**로컬 AI 데이터 분석 작업 공간 (Local AI Data Operating Room)**

실행 가능한 템플릿, 지능형 AI, DuckDB OLAP, 감사 가능한 셀, 구조화된 시각적 언어 및 검사 플라이휠을 안전하고 정교한 데스크톱 소프트웨어에 통합하여 로컬 환경에서 데이터 분석 성능을 극대화합니다.

## 긱(Geek)의 핵심 원칙

- **100% 로컬 저장소 (개인정보 보호)** — 데이터는 절대 로컬 환경을 벗어나지 않습니다. 귀하의 개인정보와 기밀 보안을 책임지며 절대적으로 안전한 개인 데이터 요새를 구축합니다.
- **제로 구성, 즉시 사용 가능 (배포 경험)** — 복잡한 배포 서비스가 필요 없습니다. Python 환경과 DuckDB 컴퓨팅 엔진이 내장되어 있어 다운로드 즉시 고밀도 분석에 돌입할 수 있습니다.
- **Harness Engineering (전체 프로세스 제어)** — 표면적인 채팅창은 버리십시오. 대규모 언어 모델은 데이터 플레인의 모든 링크에 깊이 통합되어 있으며, 통제되고 감사 가능한 프로세스 하에서 노드 검사 및 보고서 생성을 주도합니다.

## 채팅창의 한계를 극복하고 로컬 데이터 분석을 재구성하다

데이터 분석에는 엄격한 계산 추론과 재사용 가능한 경험이 필요합니다. SanchoExo는 분석 프로세스를 실행 가능하고, 추적 가능하며, 축적 가능한 로컬 유기적 시스템으로 변환합니다.

### 01 / PROCESS SYSTEM - 실행 가능한 템플릿 시스템
SanchoExo의 템플릿은 분석 경험을 실행 가능하게 캡슐화한 것입니다. 챕터 구조, 기준 제약 조건 및 체크포인트를 포함하는 뼈대를 구축하고, AI가 뼈대를 따라 셀 계산을 작성합니다. 이를 통해 제로에서 시작하는 혼란과 작별하고 방법론을 직접 구현할 수 있습니다.

### 02 / AGENT INTEGRATION - 지능형 분석 지원
Queen은 프로젝트, 노트북 셀 단위, 로컬 지식 기반 및 도구를 완벽하게 이해하고, 지속적인 후속 질문, 심층 검사 및 복잡한 데이터 보고서 생성을 지원합니다.

<img src="../../website/img/app-notebook-dark.png" alt="Unified Notebook" width="100%">

### 03 / PERFORMANCE CORE - 초고속 DuckDB OLAP
경량화된 컬럼형 컴퓨팅 코어 내장. 서비스 배포 없이 수백만 개의 로컬 CSV, Excel 및 데이터베이스에 직접 연결합니다. 극도로 빠른 응답성으로 데이터 이동 비용을 없앱니다.

### 04 / EXPERIENCE RECYCLE - 분석 경험 환원 플라이휠
노트북 셀, 데이터 컨텍스트, 검사 이상 현상 및 시각적 선호도는 분석 중에 자동으로 축적됩니다. 가치 있는 방법론은 지속적으로 추출되어 템플릿으로 환원되며, 소프트웨어를 사용할수록 더 똑똑해지고 사용자의 분석 습관을 더 깊이 이해하게 됩니다.

<img src="../../website/img/app-data-inspection.png" alt="Data Inspection" width="100%">

### 05 / VISUAL LANGUAGE - SanchoVis 시각화
구조화된 지침을 사용하여 셀은 극도로 안정적인 예상치로 차트를 생성하고, 이를 자동으로 SanchoVis 렌더러에 전달하여 전문적인 분석 기준을 완벽하게 재현할 수 있습니다.

### 06 / DESKTOP COMPACT - 원클릭 실행 환경
즉시 사용 가능. Python 종속성, SQL 실행기, 브라우저 자동화 프록시 및 지식 기반 처리가 런타임에 깊이 통합되어 있어 번거로운 환경 설정에 작별을 고할 수 있습니다.

### 07 / GLOBAL REACH - 기본 다국어 환경
글로벌 18개 언어 인터페이스의 원활한 전환을 기본적으로 지원하여, 어디에 있든 가장 익숙한 모국어 환경에서 원활하게 심층 분석을 수행할 수 있습니다.
지원 언어: 중국어(간체/번체), 영어, 일본어, 한국어, 스페인어, 프랑스어, 독일어, 러시아어, 포르투갈어, 아랍어, 힌디어, 인도네시아어, 태국어, 베트남어, 터키어, 폴란드어, 히브리어.

## 분석 스토리 프로세스

대부분의 도구는 일부 분석 단계에만 개입합니다. SanchoExo는 가져오기, 계산, 시각화, 검사, 후속 질문 및 축적/공유를 동일한 로컬 파이프라인에 완벽하게 연결합니다.

1. **원활한 데이터 접근**: 로컬 Excel, CSV 보고서, 외부 비즈니스 데이터베이스 및 웹 페이지의 구조화된 추출은 100% 로컬에 유지되며, 지연 없이 동일한 보안 분석 공간에 들어갑니다.
2. **극한의 관계형 계산**: 셀은 직접 계산 단위 역할을 하며 DuckDB의 초고속 지원을 받아 중간 결과가 지속적이고 매우 안전하게 다음 계산 단계로 흐릅니다.
3. **정확한 의도 표현**: SanchoVis 시각화 명령은 차트 렌더링을 고도로 표준화하여 경박함을 버리고 매우 읽기 쉽고 결정론적인 기준 정렬 기능을 갖추고 있습니다.
4. **능동적 검사 및 추적**: AI 엔진은 노드 및 링크 토폴로지 그래프를 기반으로 데이터 이상을 자동으로 검사하고 핵심 통찰력을 적극적으로 제안하며 추론 프로세스를 철저히 추적하고 재생합니다.
5. **폐쇄 루프 경험 추출**: 가치 있는 결론은 노트북으로 영구적으로 저장되며 반복되는 분석 프로세스는 지속적으로 개선되어 플라이휠로 환원되어 다음 작업의 디딤돌로 변환됩니다.
