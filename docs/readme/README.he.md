# SanchoExo

**סביבת עבודה מקומית לניתוח נתוני בינה מלאכותית (Local AI Data Operating Room)**

שלב תבניות ניתנות להפעלה, בינה מלאכותית חכמה, DuckDB OLAP, תאים הניתנים לביקורת, שפות חזותיות מובנות וגלגלי תנופה לבדיקה בתוכנת שולחן עבודה מאובטחת ומעודנת, המאפשרת לניתוח נתונים לשחרר את הביצועים שלו באופן מקומי.

## עקרונות ליבה של גיקים

- **100% אחסון מקומי (גיק פרטיות)** — נתונים לעולם אינם עוזבים את המחשב המקומי שלך. אנו אחראים לפרטיותך ולאבטחה חסויה, ובונים מבצר נתונים אישיים מאובטח לחלוטין.
- **אפס תצורה, מוכן לשימוש מהקופסה (חווית פריסה)** — אין צורך בשירותי פריסה מורכבים. סביבת פייתון מובנית ומנוע חישוב DuckDB. הורד וצלול מיד לניתוח בצפיפות גבוהה.
- **Harness Engineering (שליטה בשרשרת השלמה)** — נטוש תיבות צ'אט שטחיות. מודלים גדולים משולבים עמוק בכל חוליה של מישור הנתונים, שולטים בבדיקות צמתים וביצירת דוחות תחת שרשרת מבוקרת וניתנת לביקורת.

## שבירת גבולות תיבות הצ'אט, עיצוב מחדש של ניתוח נתונים מקומי

ניתוח נתונים דורש חשיבה חישובית קפדנית וניסיון לשימוש חוזר. SanchoExo הופך את תהליך הניתוח למערכת אורגנית מקומית הניתנת להפעלה, ניתנת למעקב וניתנת לשקיעה.

### 01 / PROCESS SYSTEM - מערכת תבניות ניתנת להפעלה
התבניות של SanchoExo הן עטיפה הניתנת להפעלה של ניסיון אנליטי. זה בונה שלד הכולל מבני פרקים, אילוצי קליבר ונקודות ביקורת. AI כותב חישובי תאים לאורך השלד, מיישם ישירות מתודולוגיות ונפרד מהבלבול של להתחיל מאפס.

### 02 / AGENT INTEGRATION - ניתוח חכם בעזרת
Queen יכולה להבין במלואה פרויקטים, יחידות תאי מחברת, מאגרי ידע מקומיים וכלים, ותומכת בשאלות המשך רציפות, בדיקות עמוקות ויצירת דוחות נתונים מורכבים.

<img src="../../website/img/app-notebook-dark.png" alt="Unified Notebook" width="100%">

### 03 / PERFORMANCE CORE - DuckDB OLAP מהיר במיוחד
ליבת חישוב עמודית קלת משקל מובנית. חיבור ישיר למיליוני קבצי CSV מקומיים, אקסל ומסדי נתונים מבלי לפרוס שירותים. תגובה מהירה במיוחד, מבטלת עלויות העברת נתונים.

### 04 / EXPERIENCE RECYCLE - גלגל תנופה להחזרת ניסיון ניתוח
תאי מחברת, הקשר נתונים, חריגות בדיקה והעדפות חזותיות שוקעים אוטומטית במהלך הניתוח. מתודולוגיות בעלות ערך רב מזוקקות ברציפות ומוחזרות לתבניות, מה שהופך את התוכנה לחכמה יותר עם השימוש ולהבנה טובה יותר של הרגלי הניתוח שלך.

<img src="../../website/img/app-data-inspection.png" alt="Data Inspection" width="100%">

### 05 / VISUAL LANGUAGE - הדמיה SanchoVis
באמצעות הוראות מובנות, תאים יכולים ליצור תרשימים עם ציפיות יציבות ביותר, ולהעביר אותם אוטומטית למרנדר SanchoVis כדי לשחזר בצורה מושלמת קליברים מקצועיים של ניתוח.

### 06 / DESKTOP COMPACT - סביבת ביצוע בלחיצה אחת
מוכן לשימוש. תלויות פייתון, מבצעי SQL, שרתי פרוקסי לאוטומציה של דפדפן ועיבוד מאגר ידע משולבים עמוק בזמן הריצה, ונפרדים מהגדרות סביבה כבדות.

### 07 / GLOBAL REACH - חוויה רב-לשונית מקורית
תומך באופן מקורי בהחלפה חלקה של 18 ממשקי שפה עולמיים, ומבטיח שבכל מקום בו אתה נמצא, תוכל לבצע ניתוח עמוק בצורה חלקה בהקשר שפת האם המוכרת לך ביותר.
השפות הנתמכות כוללות: סינית פשוטה, סינית מסורתית, אנגלית, יפנית, קוריאנית, ספרדית, צרפתית, גרמנית, רוסית, פורטוגזית, ערבית, הינדי, אינדונזית, תאילנדית, וייטנאמית, טורקית, פולנית ועברית.

## שרשרת סיפור הניתוח

רוב הכלים מתערבים רק בשלבי ניתוח חלקיים. SanchoExo מחבר בצורה חלקה ייבוא, חישוב, הדמיה, בדיקה, שאלות המשך ושקיעה/שיתוף על אותו פס ייצור מקומי.

1. **גישה חלקה לנתונים**: דוחות אקסל, CSV מקומיים, מסדי נתונים עסקיים חיצוניים וחילוץ מובנה של דפי אינטרנט נשמרים באופן מקומי ב-100%, ונכנסים לאותו מרחב ניתוח מאובטח ללא עיכוב.
2. **חישוב יחסי קיצוני**: תאים פועלים ישירות כיחידות חישוב, הנתמכות במהירות גבוהה במיוחד על ידי DuckDB. תוצאות ביניים זורמות באופן רציף ובטוח במיוחד לשלב החישוב הבא.
3. **הבעת כוונה מדויקת**: הוראות ההדמיה של SanchoVis מתקננות באופן גבוה יצירת תרשימים, נוטשות את האימפולסיביות ובעלות יכולות יישור קליבר קריאות ודטרמיניסטיות במיוחד.
4. **בדיקה ומעקב יזומים**: מנוע ה-AI בודק אוטומטית אנומליות נתונים בהתבסס על גרף הטופולוגיה של הצמתים והקישורים, מציע באופן יזום תובנות ליבה, ועוקב ומשחזר ביסודיות את תהליך ההסקה.
5. **זיקוק ניסיון במעגל סגור**: מסקנות בעלות ערך גבוה מאוחסנות בהתמדה כמחברות. תהליכי ניתוח שחוזרים על עצמם משתכללים ברציפות ומוזנים בחזרה לגלגל התנופה, והופכים לאבני מדרך למשימה הבאה.
