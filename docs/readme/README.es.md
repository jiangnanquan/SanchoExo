# SanchoExo

**Espacio de trabajo local para el análisis de datos con IA (Local AI Data Operating Room)**

Integre plantillas ejecutables, IA inteligente, DuckDB OLAP, celdas auditables, lenguajes visuales estructurados y bucles de inspección en un software de escritorio seguro y refinado, permitiendo que el análisis de datos desate su rendimiento localmente.

## Principios Básicos de los Geeks

- **100% de almacenamiento local (Fanático de la privacidad)** — Los datos nunca salen de su máquina local. Somos responsables de su privacidad y seguridad confidencial, construyendo una fortaleza de datos personales absolutamente segura.
- **Cero configuración, listo para usar (Experiencia de implementación)** — No hay necesidad de servicios de implementación complejos. Entorno Python integrado y motor de cálculo DuckDB. Descargue y sumérjase de inmediato en el análisis de alta densidad.
- **Harness Engineering (Control de toda la cadena)** — Abandone los cuadros de chat superficiales. Los modelos grandes están profundamente orquestados en cada enlace del plano de datos, dominando las inspecciones de nodos y la generación de informes bajo una cadena controlada y auditable.

## Rompiendo los límites de los cuadros de chat, remodelando el análisis de datos local

El análisis de datos requiere un razonamiento computacional riguroso y experiencia reutilizable. SanchoExo transforma el proceso de análisis en un sistema orgánico local ejecutable, rastreable y que se puede precipitar.

### 01 / PROCESS SYSTEM - Sistema de Plantillas Ejecutables
Las plantillas de SanchoExo son la encapsulación ejecutable de la experiencia analítica. Construye un esqueleto que incluye estructuras de capítulos, restricciones de calibre y puntos de control. La IA escribe cálculos de celda a lo largo del esqueleto, implementando directamente metodologías y diciendo adiós a la confusión de comenzar desde cero.

### 02 / AGENT INTEGRATION - Análisis Asistido Inteligente
Queen puede comprender completamente proyectos, unidades de celdas de portátiles, bases de conocimiento locales y herramientas, apoyando preguntas de seguimiento continuas, inspecciones profundas y la generación de informes de datos complejos.

<img src="../../website/img/app-notebook-dark.png" alt="Unified Notebook" width="100%">

### 03 / PERFORMANCE CORE - DuckDB OLAP ultrarrápido
Núcleo de cálculo columnar ligero incorporado. Conexión directa a millones de CSV, Excel y bases de datos locales sin implementar servicios. Respuesta extremadamente rápida, eliminando los costos de movimiento de datos.

### 04 / EXPERIENCE RECYCLE - Volante de Retorno de Experiencia de Análisis
Las celdas del cuaderno, el contexto de los datos, las anomalías de inspección y las preferencias visuales se precipitan automáticamente durante el análisis. Las metodologías de alto valor se destilan continuamente y se devuelven a las plantillas, lo que hace que el software sea más inteligente con el uso y entienda mejor sus hábitos de análisis.

<img src="../../website/img/app-data-inspection.png" alt="Data Inspection" width="100%">

### 05 / VISUAL LANGUAGE - Visualización SanchoVis
Usando instrucciones estructuradas, las celdas pueden generar gráficos con expectativas extremadamente estables, entregándolos automáticamente al renderizador SanchoVis para reproducir perfectamente los calibres de análisis profesional.

### 06 / DESKTOP COMPACT - Entorno de Ejecución de un Clic
Listo para usar. Las dependencias de Python, los ejecutores de SQL, los proxies de automatización del navegador y el procesamiento de la base de conocimientos están profundamente integrados en tiempo de ejecución, diciendo adiós a las configuraciones de entorno pesadas.

### 07 / GLOBAL REACH - Experiencia multilingüe nativa
Admite de forma nativa el cambio continuo de interfaces en 18 idiomas globales, asegurando que dondequiera que esté, pueda realizar análisis profundos sin problemas en el contexto de su lengua materna más familiar.
Los idiomas admitidos incluyen: chino simplificado, chino tradicional, inglés, japonés, coreano, español, francés, alemán, ruso, portugués, árabe, hindi, indonesio, tailandés, vietnamita, turco, polaco y hebreo.

## La cadena de la historia del análisis

La mayoría de las herramientas solo intervienen en pasos de análisis parciales. SanchoExo vincula perfectamente la importación, el cálculo, la visualización, la inspección, el seguimiento y la precipitación/intercambio en la misma línea de montaje local.

1. **Acceso fluido a los datos**: los informes locales de Excel, CSV, las bases de datos comerciales externas y la extracción estructurada de páginas web se retienen al 100% localmente, ingresando al mismo espacio de análisis seguro sin demoras.
2. **Cálculo relacional extremo**: las celdas actúan directamente como unidades de cálculo, respaldadas a una velocidad ultra alta por DuckDB. Los resultados intermedios fluyen de forma continua y altamente segura hacia el siguiente paso de cálculo.
3. **Expresión precisa de la intención**: las instrucciones de visualización de SanchoVis estandarizan en gran medida la representación de gráficos, abandonando la impetuosidad y poseyendo capacidades de alineación de calibre altamente legibles y deterministas.
4. **Inspección proactiva y seguimiento**: el motor de inteligencia artificial inspecciona automáticamente las anomalías de datos según el gráfico de topología de nodos y enlaces, propone proactivamente conocimientos fundamentales y rastrea a fondo y reproduce el proceso de razonamiento.
5. **Destilación de la experiencia de ciclo cerrado**: las conclusiones de alto valor se almacenan de forma persistente como Notebooks. Los procesos de análisis repetitivos se refinan continuamente y se retroalimentan al volante, convirtiéndose en escalones para la próxima tarea.
