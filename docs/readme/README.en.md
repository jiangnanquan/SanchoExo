## Core Geek Principles

- **100% Local Storage (Privacy Geek)** — Data absolutely never leaves your local machine. We are responsible for your privacy and confidential security, building an absolutely secure personal data fortress.
- **Zero Config, Out of the Box (Deployment Experience)** — No need for complex deployment services. Built-in Python environment and DuckDB execution engine. Download and immediately dive into high-density analysis.
- **Harness Engineering (Full-Chain Mastery)** — Abandon superficial chatboxes. Large models are deeply orchestrated into every link of the data plane, dominating node inspections and report generation under a controlled, auditable chain.

## Breaking the Limits of Chatboxes, Reshaping Local Data Analysis

Data analysis requires rigorous computational reasoning and reusable experience. SanchoExo transforms the analysis process into an executable, traceable, and precipitable local organic system.

### 01 / PROCESS SYSTEM - Executable Template System
SanchoExo's templates are the executable encapsulation of analytical experience. It builds a skeleton including chapter structures, caliber constraints, and checkpoints. AI writes Cell calculations along the skeleton, directly implementing methodologies and saying goodbye to the confusion of starting from scratch.

### 02 / AGENT INTEGRATION - Intelligent Assisted Analysis
Queen can fully understand projects, notebook Cell units, local knowledge bases, and tools, supporting continuous follow-up questions, deep inspections, and the generation of complex data reports.

<img src="../../website/img/app-notebook-dark.png" alt="Unified Notebook" width="100%">

### 03 / PERFORMANCE CORE - Ultra-fast DuckDB OLAP
Built-in lightweight columnar calculation core. Direct connection to millions of local CSVs, Excels, and databases without deploying services. Extremely fast response, eliminating data moving costs.

### 04 / EXPERIENCE RECYCLE - Analysis Experience Return Flywheel
Notebook Cells, data context, inspection anomalies, and visual preferences are automatically precipitated during analysis. High-value methodologies are continuously distilled and returned to templates, making the software smarter with use and better understanding your analysis habits.

<img src="../../website/img/app-data-inspection.png" alt="Data Inspection" width="100%">

### 05 / VISUAL LANGUAGE - SanchoVis Visualization
Using structured instructions, Cells can generate charts with extremely stable expectations, automatically handing them over to the SanchoVis renderer to perfectly reproduce professional analysis calibers.

### 06 / DESKTOP COMPACT - One-Click Execution Environment
Out of the box. Python dependencies, SQL executors, browser automation proxies, and knowledge base processing are deeply integrated at runtime, saying goodbye to heavy environment setups.

### 07 / GLOBAL REACH - Native Multi-Language Experience
Natively supports seamless switching of 18 global language interfaces, ensuring that wherever you are, you can smoothly conduct deep analysis in your most familiar native context.
Supported languages include: Simplified Chinese, Traditional Chinese, English, Japanese, Korean, Spanish, French, German, Russian, Portuguese, Arabic, Hindi, Indonesian, Thai, Vietnamese, Turkish, Polish, and Hebrew.

## The Analysis Story Chain

Most tools only intervene in partial analysis steps. SanchoExo seamlessly links import, calculation, visualization, inspection, follow-up, and precipitation/sharing onto the same local assembly line.

1. **Smooth Data Access**: Local Excel, CSV reports, external business databases, and structured web page extraction are 100% retained locally, entering the same secure analysis space with zero delay.
2. **Extreme Relational Calculation**: Cells directly act as calculation units, supported at ultra-high speed by DuckDB. Intermediate results flow continuously and highly securely to the next calculation step.
3. **Precise Intent Expression**: SanchoVis visualization instructions highly standardize chart rendering, abandoning impetuousness, and possessing highly readable and deterministic caliber alignment capabilities.
4. **Proactive Inspection & Follow-up**: The AI engine automatically inspects data anomalies based on the node and link topology graph, proactively proposes core insights, and thoroughly traces and replays the reasoning process.
5. **Closed-Loop Experience Distillation**: High-value conclusions are persistently stored as Notebooks. Repetitive analysis processes are continuously refined and fed back into the flywheel, turning into stepping stones for the next task.
