# SanchoExo

**ローカル AI データ分析ワークスペース (Local AI Data Operating Room)**

実行可能なテンプレート、インテリジェントな AI、DuckDB OLAP、監査可能なセル、構造化された視覚的言語、検査フライホイールを安全で洗練されたデスクトップソフトウェアに統合し、ローカル環境でデータ分析のパフォーマンスを最大限に引き出します。

[公式サイト (sanchoexo.com)](https://sanchoexo.com) · [ダウンロード](#ダウンロード) · [機能](#チャットボックスの限界を打ち破りローカルデータ分析を再構築) · [フィードバック](https://github.com/jiangnanquan/SanchoExo/issues)

## ギークのコア原則

- **100% ローカルストレージ（プライバシーの追求）** — データは絶対にローカルから出ません。お客様のプライバシーと機密性のセキュリティに責任を持ち、完全に安全な個人データ要塞を構築します。
- **ゼロコンフィグ、設定不要（デプロイメント体験）** — 複雑なデプロイメントサービスは不要です。Python 環境と DuckDB 計算エンジンを内蔵。ダウンロードしてすぐに高密度な分析に没頭できます。
- **Harness Engineering（全プロセスの制御）** — 表面的なチャットボックスを捨てましょう。大規模言語モデルはデータプレーンのあらゆるリンクに深く組み込まれており、制御可能で監査可能なプロセスの下で、ノードの検査とレポート生成を主導します。

## チャットボックスの限界を打ち破り、ローカルデータ分析を再構築

データ分析には、厳密な計算推論と再利用可能な経験が必要です。SanchoExo は、分析プロセスを実行可能で、追跡可能で、蓄積可能なローカルの有機的システムに変換します。

### 01 / PROCESS SYSTEM - 実行可能テンプレートシステム
SanchoExo のテンプレートは、分析の経験を実行可能にカプセル化したものです。章の構造、基準の制約、チェックポイントを含むスケルトンを構築し、AI がそのスケルトンに沿ってセル計算を記述します。これにより、ゼロから始める迷いに別れを告げ、方法論を直接実装できます。

### 02 / AGENT INTEGRATION - インテリジェントな分析支援
Queen はプロジェクト、ノートブックのセル単位、ローカルナレッジベース、およびツールを完全に理解し、継続的なフォローアップの質問、綿密な検査、および複雑なデータレポートの生成をサポートします。

<img src="../../website/img/app-notebook-dark.png" alt="Unified Notebook" width="100%">

### 03 / PERFORMANCE CORE - 超高速 DuckDB OLAP
軽量の列指向計算コアを内蔵。サービスをデプロイすることなく、数百万行のローカル CSV、Excel、およびデータベースに直接接続します。極めて高速な応答により、データ移動のコストを排除します。

### 04 / EXPERIENCE RECYCLE - 分析経験の還元フライホイール
ノートブックのセル、データのコンテキスト、検査の異常、および視覚的な好みは、分析中に自動的に蓄積されます。価値の高い方法論は継続的に抽出されてテンプレートに還元され、ソフトウェアを使用するほどに賢くなり、ユーザーの分析習慣をより深く理解するようになります。

<img src="../../website/img/app-data-inspection.png" alt="Data Inspection" width="100%">

### 05 / VISUAL LANGUAGE - SanchoVis ビジュアライゼーション
構造化された指示を使用して、セルは極めて安定した期待値でグラフを生成し、それを SanchoVis レンダラーに自動的に渡して、専門的な分析基準を完璧に再現できます。

### 06 / DESKTOP COMPACT - ワンクリック実行環境
設定不要。Python の依存関係、SQL エグゼキューター、ブラウザー自動化プロキシ、およびナレッジベースの処理がランタイムに深く統合されており、面倒な環境設定に別れを告げることができます。

### 07 / GLOBAL REACH - ネイティブな多言語エクスペリエンス
世界 18 言語のインターフェースのシームレスな切り替えをネイティブでサポートし、どこにいても最も使い慣れた母国語のコンテキストで深い分析をスムーズに行うことができます。
サポートされている言語: 簡体字中国語、繁体字中国語、英語、日本語、韓国語、スペイン語、フランス語、ドイツ語、ロシア語、ポルトガル語、アラビア語、ヒンディー語、インドネシア語、タイ語、ベトナム語、トルコ語、ポーランド語、ヘブライ語。

## 分析のストーリープロセス

ほとんどのツールは、一部の分析ステップにのみ介入します。SanchoExo は、インポート、計算、視覚化、検査、フォローアップの質問、および蓄積と共有を、同じローカルのパイプラインにシームレスにリンクします。

1. **スムーズなデータアクセス**: ローカルの Excel、CSV レポート、外部ビジネスデータベース、および Web ページの構造化された抽出は 100% ローカルに保持され、遅延なしで同じ安全な分析スペースに入ります。
2. **極限の関連計算**: セルは直接計算単位として機能し、DuckDB によって超高速でサポートされ、中間結果は継続的かつ非常に安全に次の計算ステップに流れます。
3. **正確な意図の表現**: SanchoVis の視覚化命令は、グラフのレンダリングを高度に標準化し、軽薄さを捨て、非常に読みやすく決定論的な基準の調整機能を備えています。
4. **プロアクティブな検査と追跡**: AI エンジンは、ノードとリンクのトポロジグラフに基づいてデータ異常を自動的に検査し、コアとなる洞察を積極的に提案し、推論プロセスを徹底的に追跡して再生します。
5. **クローズドループの経験抽出**: 価値の高い結論はノートブックとして永続的に保存され、繰り返される分析プロセスは継続的に洗練されてフライホイールに還元され、次のタスクの足がかりに変換されます。
