# SanchoExo

**ห้องปฏิบัติการข้อมูล AI ในพื้นที่ (Local AI Data Operating Room)**

รวมเทมเพลตที่สามารถดำเนินการได้, AI อัจฉริยะ, DuckDB OLAP, เซลล์ที่ตรวจสอบได้, ภาษาภาพที่มีโครงสร้าง และฟลายวีลการตรวจสอบเข้ากับซอฟต์แวร์เดสก์ท็อปที่ปลอดภัยและประณีต ทำให้การวิเคราะห์ข้อมูลสามารถปลดปล่อยประสิทธิภาพในพื้นที่ได้

[เว็บไซต์อย่างเป็นทางการ (sanchoexo.com)](https://sanchoexo.com) · [ดาวน์โหลด](#ดาวน์โหลด) · [คุณสมบัติ](#ทลายขีดจำกัดของกล่องสนทนา-พลิกโฉมการวิเคราะห์ข้อมูลในพื้นที่) · [ข้อเสนอแนะ](https://github.com/jiangnanquan/SanchoExo/issues)

## หลักการสำคัญของ Geek

- **พื้นที่เก็บข้อมูลในพื้นที่ 100% (Privacy Geek)** — ข้อมูลจะไม่ออกจากเครื่องของคุณโดยเด็ดขาด เรามีหน้าที่รับผิดชอบต่อความเป็นส่วนตัวและความปลอดภัยที่เป็นความลับของคุณ สร้างป้อมปราการข้อมูลส่วนบุคคลที่ปลอดภัยอย่างยิ่ง
- **ไม่ต้องกำหนดค่า พร้อมใช้งาน (ประสบการณ์การปรับใช้)** — ไม่จำเป็นต้องมีบริการการปรับใช้ที่ซับซ้อน สภาพแวดล้อม Python ในตัวและกลไกการคำนวณ DuckDB ดาวน์โหลดและเจาะลึกการวิเคราะห์ความหนาแน่นสูงได้ทันที
- **Harness Engineering (การควบคุมทั้งเครือข่าย)** — ละทิ้งกล่องสนทนาที่ผิวเผิน โมเดลภาษาขนาดใหญ่ได้รับการจัดระเบียบอย่างลึกซึ้งในทุกๆ ลิงก์ของดาต้าเพลน โดยครอบงำการตรวจสอบโหนดและการสร้างรายงานภายใต้เครือข่ายที่ควบคุมและตรวจสอบได้

## ทลายขีดจำกัดของกล่องสนทนา พลิกโฉมการวิเคราะห์ข้อมูลในพื้นที่

การวิเคราะห์ข้อมูลต้องการการให้เหตุผลทางการคำนวณที่เข้มงวดและประสบการณ์ที่นำมาใช้ซ้ำได้ SanchoExo แปลงกระบวนการวิเคราะห์ให้เป็นระบบอินทรีย์ในพื้นที่ที่สามารถดำเนินการได้ ตรวจสอบย้อนกลับได้ และตกตะกอนได้

### 01 / PROCESS SYSTEM - ระบบเทมเพลตที่สามารถดำเนินการได้
เทมเพลตของ SanchoExo คือการห่อหุ้มประสบการณ์การวิเคราะห์ที่สามารถดำเนินการได้ มันสร้างโครงกระดูกซึ่งรวมถึงโครงสร้างบท ข้อจำกัดด้านความสามารถ และจุดตรวจสอบ AI เขียนการคำนวณเซลล์ตามโครงกระดูก นำวิธีการไปใช้โดยตรง และบอกลาความสับสนของการเริ่มต้นจากศูนย์

### 02 / AGENT INTEGRATION - การวิเคราะห์ด้วยความช่วยเหลืออัจฉริยะ
Queen สามารถเข้าใจโครงการ หน่วยเซลล์สมุดบันทึก ฐานความรู้ในพื้นที่ และเครื่องมือต่างๆ ได้อย่างถ่องแท้ รองรับคำถามติดตามผลอย่างต่อเนื่อง การตรวจสอบเชิงลึก และการสร้างรายงานข้อมูลที่ซับซ้อน

<img src="./website/img/app-notebook-dark.png" alt="Unified Notebook" width="100%">

### 03 / PERFORMANCE CORE - DuckDB OLAP ที่รวดเร็วเป็นพิเศษ
แกนการคำนวณแบบคอลัมน์น้ำหนักเบาในตัว เชื่อมต่อโดยตรงกับ CSV, Excel และฐานข้อมูลในพื้นที่หลายล้านรายการโดยไม่ต้องปรับใช้บริการ ตอบสนองอย่างรวดเร็ว ขจัดค่าใช้จ่ายในการย้ายข้อมูล

### 04 / EXPERIENCE RECYCLE - ฟลายวีลคืนประสบการณ์การวิเคราะห์
เซลล์สมุดบันทึก บริบทข้อมูล ความผิดปกติในการตรวจสอบ และการตั้งค่าการแสดงผลจะถูกตกตะกอนโดยอัตโนมัติระหว่างการวิเคราะห์ วิธีการที่มีมูลค่าสูงจะถูกกลั่นและส่งคืนไปยังเทมเพลตอย่างต่อเนื่อง ทำให้ซอฟต์แวร์ฉลาดขึ้นเมื่อใช้งานและเข้าใจพฤติกรรมการวิเคราะห์ของคุณได้ดียิ่งขึ้น

<img src="./website/img/app-data-inspection.png" alt="Data Inspection" width="100%">

### 05 / VISUAL LANGUAGE - การสร้างภาพ SanchoVis
ด้วยการใช้คำสั่งที่มีโครงสร้าง เซลล์สามารถสร้างแผนภูมิด้วยความคาดหวังที่มั่นคงเป็นอย่างยิ่ง โดยอัตโนมัติส่งมอบให้กับผู้เรนเดอร์ SanchoVis เพื่อสร้างความสามารถในการวิเคราะห์ระดับมืออาชีพได้อย่างสมบูรณ์แบบ

### 06 / DESKTOP COMPACT - สภาพแวดล้อมการทำงานในคลิกเดียว
พร้อมใช้งาน การพึ่งพา Python ผู้ดำเนินการ SQL พร็อกซีการทำงานอัตโนมัติของเบราว์เซอร์ และการประมวลผลฐานความรู้ถูกรวมเข้าด้วยกันอย่างลึกซึ้งในขณะใช้งาน บอกลาการตั้งค่าสภาพแวดล้อมที่หนักหน่วง

### 07 / GLOBAL REACH - ประสบการณ์หลายภาษาพื้นเมือง
รองรับการสลับอินเทอร์เฟซภาษาทั่วโลก 18 ภาษาได้อย่างราบรื่น เพื่อให้มั่นใจว่าไม่ว่าคุณจะอยู่ที่ไหน คุณสามารถทำการวิเคราะห์เชิงลึกได้อย่างราบรื่นในบริบทภาษาแม่ที่คุณคุ้นเคยที่สุด
ภาษาที่รองรับได้แก่: จีนตัวย่อ จีนตัวเต็ม อังกฤษ ญี่ปุ่น เกาหลี สเปน ฝรั่งเศส เยอรมัน รัสเซีย โปรตุเกส อาหรับ ฮินดี อินโดนีเซีย ไทย เวียดนาม ตุรกี โปแลนด์ และฮีบรู

## ห่วงโซ่เรื่องราวการวิเคราะห์

เครื่องมือส่วนใหญ่จะแทรกแซงเฉพาะในขั้นตอนการวิเคราะห์บางส่วนเท่านั้น SanchoExo เชื่อมโยงการนำเข้า การคำนวณ การแสดงภาพ การตรวจสอบ คำถามติดตามผล และการตกตะกอน/การแบ่งปันในสายการประกอบในพื้นที่เดียวกันได้อย่างราบรื่น

1. **การเข้าถึงข้อมูลที่ราบรื่น**: รายงาน Excel, CSV ในพื้นที่ ฐานข้อมูลธุรกิจภายนอก และการดึงข้อมูลหน้าเว็บที่มีโครงสร้างจะถูกเก็บไว้ในเครื่อง 100% เข้าสู่พื้นที่การวิเคราะห์ที่ปลอดภัยแบบเดียวกันโดยไม่ล่าช้า
2. **การคำนวณเชิงสัมพันธ์ขั้นสุด**: เซลล์ทำหน้าที่เป็นหน่วยการคำนวณโดยตรง ซึ่งรองรับด้วยความเร็วสูงพิเศษโดย DuckDB ผลลัพธ์ระดับกลางจะไหลไปสู่ขั้นตอนการคำนวณถัดไปอย่างต่อเนื่องและปลอดภัยสูง
3. **การแสดงความตั้งใจที่แม่นยำ**: คำสั่งการแสดงผลของ SanchoVis กำหนดมาตรฐานการแสดงผลแผนภูมิในระดับสูง ละทิ้งความใจร้อน และมีความสามารถในการจัดตำแหน่งความสามารถในการอ่านและการกำหนดค่าสูง
4. **การตรวจสอบและติดตามเชิงรุก**: กลไก AI จะตรวจสอบความผิดปกติของข้อมูลโดยอัตโนมัติตามกราฟทอพอโลยีของโหนดและลิงก์ เสนอข้อมูลเชิงลึกหลักในเชิงรุก และติดตามและเล่นกระบวนการให้เหตุผลซ้ำอย่างละเอียด
5. **การกลั่นกรองประสบการณ์แบบปิด**: ข้อสรุปที่มีมูลค่าสูงจะถูกจัดเก็บไว้อย่างถาวรเป็นสมุดบันทึก กระบวนการวิเคราะห์ที่ทำซ้ำจะได้รับการปรับปรุงอย่างต่อเนื่องและป้อนกลับไปที่ฟลายวีล เปลี่ยนเป็นก้าวสำคัญสำหรับงานต่อไป
